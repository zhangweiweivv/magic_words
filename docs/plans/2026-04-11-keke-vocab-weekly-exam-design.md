# 可可单词网站 - 周考功能设计

> 创建日期：2026-04-11
> 状态：已批准

---

## 背景

可可单词网站现有的测试功能嵌在艾宾浩斯复习流程中（Day 0/1/3 测试），单词毕业到「已记住」后就不再考了。需要一个独立的周考功能，定期验证已毕业单词的掌握情况。

## 功能概述

每周六自动生成一套考试，验证最近 N 周内毕业的已记住单词。考试在下次生成前始终可做。

---

## 考试触发

- **生成时机**：每周六（Cron 任务，可在家长中心修改考试日）
- **有效期**：周六生成 → 下周五（下次考试生成时替换）
- **首页入口**：
  - 未完成：显示考试入口卡片（类似"每日探险任务"）
  - 已完成：显示"查看成绩"
  - 无考试/已过期：不显示
- **Slack 提醒**：考试生成时推送到 #可可pet

---

## 考试范围

仅限最近 N 周内毕业的「已记住」单词。

- N 默认 = 1（即过去7天内毕业的词）
- N 可在家长中心配置

---

## 出题规则

在 N 周窗口内的已毕业单词中：

### 1. 错题（100% 纳入）

该窗口内，历次周考答错过的单词，全部纳入考试。

- 答错记录存储在 `周考错题池.json`
- 窗口外的错题不纳入（即 N 周前毕业的词即使答错过也不再考）

### 2. 非错题（按难度抽样）

窗口内、不在错题池中的单词，按 CEFR 难度等级随机抽样：

| 难度 | 默认抽样率 |
|------|----------|
| A1   | 20%      |
| A2   | 40%      |
| B1+  | 60%      |

抽样率可在家长中心调整。

### 3. 难度识别

- 查询 `PET官方单词库.md`，复用扩充脚本的解析逻辑（按 🏷️ 标题识别等级）
- 查不到的单词默认归为 A2

### 4. 题量

自适应——错题全部 + 抽样结果，有多少考多少。

---

## 题型配比

按单词难度分配题型：

| 难度 | 选择题 | 补全题 | 拼写题 |
|------|--------|--------|--------|
| A1   | 60%    | 30%    | 10%    |
| A2   | 30%    | 40%    | 30%    |
| B1+  | 10%    | 30%    | 60%    |
| 错题 | 0%     | 0%     | 100%   |

复用现有 QuizMode 的三种题型组件。

---

## 考试流程

```
首页点击"周考试卷" 
    ↓
显示考试介绍（题量、范围说明）
    ↓
开始答题（逐题作答，显示进度条）
    ↓
第一轮完成 → 记录成绩（正确率）
    ↓
有错题？→ 错题循环重做（只做错的）→ 全对
    ↓
考试通过 → 显示成绩单 → 记录结果
    ↓
首页入口变为"查看成绩"
```

### 成绩计算

- **得分 = 第一轮正确率**（如 15题对12题 = 80分）
- 错题重做不影响分数，仅用于强化记忆
- 第一轮答错的单词记入错题池，下次周考 100% 再出

## 数据存储

### 周考记录（`可可单词本/周考记录.md`）

```markdown
> [!exam] 📅 2026-04-12 周考
> - 总题数：18
> - 第一轮正确：15 / 18（83分）
> - 错题重做轮数：2
> - 错题单词：courage, equipment, integrity
> - 考试范围：最近1周毕业词
```

### 周考错题池（`可可单词本/周考错题池.json`）

```json
{
  "errors": {
    "courage": {
      "wrongDates": ["2026-04-12"],
      "graduatedDate": "2026-04-07"
    }
  }
}
```

### 周考状态（`可可单词本/.weekly-exam-status.json`）

```json
{
  "currentExam": {
    "generatedDate": "2026-04-12",
    "words": [...],
    "completed": false,
    "score": null
  },
  "config": {
    "examDay": 6,
    "windowWeeks": 1,
    "sampleRates": { "A1": 0.2, "A2": 0.4, "B1+": 0.6 }
  }
}
```

## 家长中心

新增"📝 周考设置"区块：

- **考试日**：周几生成（默认周六）
- **考试窗口 N**：最近几周毕业的词（默认1周）
- **抽样率**：A1 / A2 / B1+ 三个滑块（0%-100%）
- **历次成绩**：日期、得分、错题列表

## Slack 通知

### 考试生成时
```
📝 本周考试已就绪！
📚 共 XX 题（错题 X + 新词 X + 抽样 X）
🎯 范围：最近1周毕业词
```

### 考试完成时
```
🎉 可可完成了本周单词考试！

📊 成绩：83分（15/18）
📝 共2轮完成

第1轮：✅ 15题 ❌ 3题
  错题：courage(拼写), equipment(补全), integrity(拼写)
第2轮：✅ 3题 ❌ 0题

💪 错题已加入下周复习池！
```
